# Pseudo_distribution-Database

## 基于Vue3, koa的伪分布式数据库管理站点

### /server & /client 安装依赖
```shell
npm install
yarn install
```

### /server & /client 运行开发环境
```shell
npm run dev
yarn dev
```

### /server & /client 项目打包
```shell
npm run build
yarn build
```

### Lints and fixes files
```shell
npm run lint 
yarn lint
```
